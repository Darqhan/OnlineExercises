  • Currently we use the HTTP2 (see http secure )
  • Sample HTTP request : (see http secure )
  • HTTP commands (see http secure )

    2. PUT
    3. DELETE

  • HTTP error codes (see http secure )
    1. 100...Informational responses
    2. 200...Success
    3. 300...Redirection codes
    4. 400...Client errors
    5. 500...Server errors

Few words about IP adress

  • An Internet Protocol address (IP address) is a numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication (see http secure )
    • versions:
    • IPv4 (32 bits) sample: 127.0.0.1
    • IPv6 (128 bits) sample: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

URL

  • Uniform Resource Locator aka web address (http, www) (see URL definition)
  • URI = scheme:[//authority]path[?query][#fragment]
  • invented by Tim BL at 1992 (they used the full path before)

So, Servers

In computing, a server is a computer program or a device that provides functionality for other programs or devices, called "clients". So the server communicates with the clients. (see more about servers
Servers by Purpose
- App (hosts web apps, using web browser)
- Catalog (directory, name)
- Database (collestion of data organized in forms)
- File (share your files)
- Print (more user connects to more printers)
- Proxy (traffic performance, authorization)
- Web (hosts online web pages)
